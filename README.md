# Project Zomboid Mods

Mods for Project Zomboid (Build 42+).

## Mods

| Mod | Description |
|-----|-------------|
| [BarricadesHurtZombiesB42](BarricadesHurtZombiesB42/) | Zombies take damage when attacking barricaded structures |
| [BarricadeContextMenu](BarricadeContextMenu/) | Restores right-click barricade/unbarricade context menu options removed in Build 42 |

## Installation

Copy the mod folder from `Contents/mods/` into your Project Zomboid mods directory:

- **Windows**: `%USERPROFILE%\Zomboid\mods\`
- **Linux**: `~/Zomboid/mods/`

Then enable the mod in-game via the Mods menu.
